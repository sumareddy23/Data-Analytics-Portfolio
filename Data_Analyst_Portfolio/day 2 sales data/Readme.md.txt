# 📊 Day 2: Regional Sales Analysis

## 🗂️ Overview
This project extends the sales performance analysis from Day 1 by focusing on **regional and salesperson-level performance**.  
Using Microsoft Excel, I explored which regions generated the highest revenue, which salespeople met their targets, and where improvement is needed.

---

## 💾 Dataset Summary
**File Used:** `day 2 sales analysis.xlsx`

| Column | Description |
|---------|-------------|
| Month | Sales month |
| Region | Business region (North, South, East, West) |
| Salesperson | Representative responsible for sales |
| Sales | Actual sales achieved |
| Target | Assigned sales target |
| Variance | Difference between Sales and Target |

---

## 🧮 Analysis Performed
- Built **Pivot Tables** summarizing:
  - Sales and Target by **Region**
  - Sales and Target by **Salesperson**
